# 엑셀 to json 컨버터

## 프롬프트
1. 리액트 프로젝트에서 i18n을 이용해서 번역 작업을하기 위한 json 파일을 생성해야해.
2. 각 언어별 json이 필요해 (ko-KR.json, en-US.json, ja-JP.json, zh-Hans.json, zh-Hant.json). 너는 그걸 컬럼 헤더명을 보고 판단하면 돼.
3. 분류: 1뎁스, 소분류: 2뎁스, 키코드: 3뎁스 키로 사용할 거야. 이거 json 으로 구조화하면 돼.
4. 나머지는 언어별 value로 설정해
5. 중요한 건 줄바꿈 정보를 읽어서 \n도 value text에 삽입되어야해.
6. 시스템언어_앱, 시스템언어_키오스크 2벌이 나와야 해 (총 10개의 json)
7. 여기에 코드로 복사할 수 있도록 적어줘
8. 언어별로 일관되게 각 요소가 명확하게 잘 들어갔는지 검증해 (총 row count 등)
7. output 폴더에 10개 파일 생성하는 스크립트
